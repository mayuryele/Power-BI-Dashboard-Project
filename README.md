# Power-BI-Dashboard-Project
End-to-end Sales & Revenue Analysis for FY 2023-2024 | Power BI Dashboard | Excel Dataset | Regional & Category Insights.

**Key Insights**
Revenue grew 21.3% YoY — from ₹95.4M in 2023 to ₹115.8M in 2024, showing strong and consistent upward momentum
West Region leads with ₹31.5M in total revenue, followed closely by North at ₹28.4M
Q4 is always the power quarter — contributes ~28% of annual revenue in both years, driven by year-end deals
Software & Services have the highest margins (55–62%), while Electronics drives the highest absolute volume
Rahul Sharma topped the sales leaderboard with ₹22.4M across both years — consistency over 24 months
Profit margin improved from 31.1% → 33.2% — a 2.1 percentage point improvement, largely driven by the shift toward software deals
Features

**Feature Details**
Time Period
Jan 2023 – Dec 2024 (24 months)
Total Records
6,847 orders
Regions
North, South, East, West, Central
Categories
Electronics, Software, Services, Hardware
KPIs Tracked
Revenue, Profit, Orders, Avg Order Value, Margin %
Dashboard Tool
Microsoft Power BI (published link)
Data Format
Excel (.xlsx) — 4 structured sheets

**Why This Project Stands Out**
Most sales analysis projects stop at pivot tables and basic charts. This one goes further:
1. Two-year comparative analysis — Not just "what happened" but "how much did it improve and why." Every metric is benchmarked against the previous year.
2. Multi-dimensional breakdown — The same revenue number is sliced by region, category, salesperson, and quarter — independently and in combination — so you can answer almost any business question without extra work.
3. Profit margin focus — Revenue is vanity, profit is sanity. Every chart pairs revenue with profit so the story stays honest.
4. Clean, audit-ready data — The Excel file uses structured headers, consistent formats, and formula-driven summary sheets. No hardcoded totals. Everything recalculates if the raw data changes.
5. Dashboard built for executives — Dark theme, KPI cards front and center, one glance = the full story. No clutter.
